# Работа с объявлениями

## <span id="advertisements">Список объявлений</span>

:::info
_GET_ `v2/advertisement/?show_deleted=false&period=today&offset=0&limit=50&advertisement_titles=&campaign_names=`
:::

### Пример использования
:::info

```http request
https://client.adstat.pro/api/v2/advertisement/?show_deleted=false&period=today&offset=0&limit=50&advertisement_titles=&campaign_names=
```
:::

### __Описание query__

:::info

| Параметр               | Тип     | Описание                                                                                                                                                                                   |
|------------------------|---------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `show_deleted`         | Boolean | Параметр, определяющий, нужно ли показывать удаленные объявления. Возможные значения: `true` или `false`.                                                                                  |
| `period`               | Строка  | Период, за который необходимо получить данные. <br/>Значения могут быть - `today` ( За сегодня ) , `all` ( За весь период ), `yesterday` (За вчера) , `starts_month` ( За текущий месяц ). |
| `offset`               | Число   | Смещение в результирующем списке объявлений.                                                                                                                                               |
| `limit`                | Число   | Максимальное количество объявлений в ответе.                                                                                                                                               |
| `advertisement_titles` | Строка  | Фильтрация по названию объявления.                                                                                                                                                         |
| `campaign_names`       | Строка  | Фильтрация по названию объявления.                                                                                                                                                         |

:::


### __Пример ответа__
:::info
```json
{
  "data": [
    {
      "id": 1364756,
      "account_name": "SOME_test",
      "campaign_name": "Default",

      "ad_text": "example text",
      "cpm": 3.0,

      "cpc": 0,

      "spent": 0,
      "clicks": 0
    }
  ]
}
```
:::

### __Описание параметров ответа__
:::info

| Поле                          | Тип          | Описание                                                                  |
|-------------------------------|--------------|---------------------------------------------------------------------------|
| `id`                          | Число        | Уникальный идентификатор объявления.                                      |
| `account_name`                | Строка       | Название аккаунта, которому принадлежит объявление.                       |
| `campaign_name`               | Строка       | Название кампании, к которой относится объявление.                        |
| `advertisement_title`         | Строка       | Название объявления.                                                      |
| `status`                      | Строка       | Текущий статус объявления.                                                |
| `object`                      | Строка       | Объект рекламирования.                                                    |
| `ad_text`                     | Строка       | Текст креатива объявления.                                                |
| `cpm`                         | Число        | Стоимость тысячи показов (CPM).                                           |
| `target_topics`               | Массив строк | Список топиков, на которые таргетируется объявление.                      |
| `balance`                     | Число        | Баланс объявлением.                                                       |
| `cps`                         | Число        | Стоимость за цель (CPS).                                                  |
| `created_dt`                  | Строка       | Дата и время создания объявления в формате строки.                        |
| `status_updated_dt`           | Строка       | Дата и время последнего обновления статуса объявления в формате ISO 8601. |
| `ad_type`                     | Строка       | Тип объявления <br/>(например, `"target_channels"`, `target_users`).      |
| `goals`                       | Число        | Количество достигнутых целей.                                             |
| `impressions`                 | Число        | Количество показов объявления.                                            |
| `cpc`                         | Число        | Стоимость за клик (CPC).                                                  |
| `ctr`                         | Число        | Коэффициент кликабельности (CTR).                                         |
| `spent`                       | Число        | Сумма потраченных средств.                                                |
| `clicks`                      | Число        | Количество кликов по объявлению.                                          |
| `successfully_created_ad_ids` | Число        | Cписок id созданых объявлений                                             |

:::

## <span id="create_advertisement">Создание объявления</span>

:::info
_POST_ `/api/advertisement/telegram/`
:::
### Пример использования:
:::info
```http request
https://client.adstat.pro/api/advertisement/telegram/
```
:::



### Описание body
:::info
В тело(`body`) HTTP запроса необходимо передать параметры, необходимые для создания объявления.

| Поле                                   | Тип                 | Описание                                                                                                                                                                |
|----------------------------------------|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `campaign_id`                          | Число               | Идентификатор кампании, к которой будет привязано объявление.                                                                                                           |
| `title`                                | Строка              | Название объявления.                                                                                                                                                    |
| `text`                                 | Строка              | Текст объявления.                                                                                                                                                       |
| `promote_url`                          | Строка              | URL объекта рекламирования.                                                                                                                                             |
| `cpm`                                  | Число               | Стоимость за 1000 показов (CPM).                                                                                                                                        |
| `budget`                               | Число               | Рекламный бюджет объявления.                                                                                                                                            |
| `langs`                                | Массив строк        | Список языков, на которых будет осуществляться таргетинг объявления.                                                                                                    |
| `topics`                               | Массив чисел        | Список тематик, на которые будет осуществляться таргетинг объявления.                                                                                                   |
| `topics_verbose`                       | Массив строк        | Список идентификаторов тематик, на которые будет осуществляться таргетинг объявления.                                                                                   |
| `channels`                             | Массив чисел        | Список каналов, на которые будет осуществляться таргетинг объявления.                                                                                                   |
| `channels_verbose`                     | Массив строк        | Список идентификаторов каналов, на которые будет осуществляться таргетинг объявления.                                                                                   |
| `exclude_topics`                       | Массив чисел        | Список исключаемых тематик, на которые не будет осуществляться таргетинг объявления.                                                                                    |
| `exclude_topics_verbose`               | Массив строк        | Список идентификаторов исключаемых тематик, на которые не будет осуществляться таргетинг объявления.                                                                    |
| `exclude_channels`                     | Массив чисел        | Список исключаемых каналов, на которые не будет осуществляться таргетинг объявления.                                                                                    |
| `exclude_channels_verbose`             | Массив строк        | Список идентификаторов исключаемых каналов, на которые не будет осуществляться таргетинг объявления.                                                                    |
| `media_token`                          | Строка              | Токен для работы с медиафайлами, если кабинету доступно использование медиа для объявлений. Данный токен возвращается метод [Загрузки медиа](#advertisement_with_media) |
| `other_info`                           | Строка              | ERID-токен. Заполнение данного поля осуществляется, если на кабинете включена ручная маркировка.                                                                        |
| `split_by`                             | Массив строк        | Список параметров для указания разбивки объявления по различным значениям таргетинга, например: ['langs'].                                                              |
| `ad_source`                            | Строка              | Источник создания объявления. Обычно передаётся строка "web_api".                                                                                                       |
| `ad_type`                              | Строка              | Тип таргетинга (например, "target_channels" или "target_users").                                                                                                        |
| `countries`                            | Массив строк        | Список стран, на которые будет осуществляться таргетинг объявления.                                                                                                     |
| `locations`                            | Массив строк        | Список городов, на которые будет осуществляться таргетинг объявления.                                                                                                   |
| `locations_verbose`                    | Массив строк        | Список идентификаторов городов, на которые будет осуществляться таргетинг объявления.                                                                                   |
| `audiences`                            | Массив чисел        | Список аудиторий, на которые будет осуществляться таргетинг объявления.                                                                                                 |
| `exclude_audiences`                    | Массив чисел        | Список исключаемой аудитории, на которую не будет осуществляться таргетинг объявления.                                                                                  |
| `website_name`                         | Строка              | Название вебсайта рекламируемого объекта. Заполняется, если рекламируемый объект является внешней ссылкой.                                                              |
| `views_per_user`                       | Число               | Дневной лимит просмотров на пользователя.                                                                                                                               |
| `show_picture`                         | Логическое значение | Указывает, показывать ли аватар в объявлении.                                                                                                                           |
| `promote_url_picture_id`               | Строка              | Идентификатор изображения, связанного с объектом рекламирования.                                                                                                        |
| `exclude_politic`                      | Логическое значение | Указывает, следует ли исключить политический контент из кампании (true/false).                                                                                          |
| `all_topics_interested_users`          | Логическое значение | Указывает, следует ли показывать объявление только пользователям с интересами во всех выбранных тематиках (true/false).                                                 |
| `device`                               | Строка              | Устройство для таргетинга (например, "all" для всех устройств).                                                                                                         |
| `target_user_channels`                 | Массив строк        | Список каналов при таргетинге на пользователей, на которые будет таргетироваться объявление.                                                                            |
| `target_user_channels_verbose`         | Массив строк        | Список идентификаторов каналов, на которые будет таргетироваться объявление при таргетинге на пользователей.                                                            |
| `exclude_target_user_channels`         | Массив строк        | Список каналов, которые следует исключить из таргетинга на пользователей.                                                                                               |
| `exclude_target_user_channels_verbose` | Массив строк        | Список идентификаторов каналов, которые следует исключить из таргетинга на пользователей.                                                                               |
| `button_type`                          | Строка              | Тип кнопки, которая будет отображаться в объявлении (например, "open_website"). Доступно только для объектов рекламирования типа "вебсайт" (внешняя ссылка).            |
| `daily_budget`                         | Число               | Дневной лимит объявления.                                                                                                                                               |
| `after_moderation_status`              | Строка              | Статус после модерации (например, "active").                                                                                                                            |
| `start_date`                           | Строка (ISO 8601)   | Дата запуска объявления в формате ISO 8601 (например, "2024-12-03T09:32:47.211Z").                                                                                      |
| `end_date`                             | Строка (ISO 8601)   | Дата остановки объявления в формате ISO 8601 (например, "2024-12-03T09:32:47.211Z").                                                                                    |
| `schedule`                             | Объект              | Расписание показа объявлений по дням недели с указанием времени.                                                                                                        |
| `use_selected_timezone`                | Логическое значение | Указывает, следует ли использовать выбранный часовой пояс. Поле передаётся при использовании расписания показа объявления.                                              |
| `schedule_timezone`                    | Число               | Часовой пояс для расписания (например, 0 для UTC).                                                                                                                      |
| `promote_url_type`                     | Строка              | Тип URL для продвижения (например, "website").                                                                                                                          |
| `currency_code`                        | Строка              | Код валюты для объявления (например, "EUR").                                                                                                                            |
| `use_account_currency`                 | Логическое значение | Указывает, использовать ли валюту аккаунта для объявления (true/false).                                                                                                 |
| `only_crypto`                          | Логическое значение | Указывает, следует ли показывать объявление только в каналах, связанных с криптовалютами (true/false).                                                                  |
| `exclude_crypto`                       | Логическое значение | Указывает, следует ли исключить показ объявления в каналах, связанных с криптовалютами (true/false).                                                                    |

:::

## <span id="advertisement_with_media">Загрузка медиа</span>

:::info
_POST_ `api/v1/advertising/telegram/upload_media?account_id={ACC_ID}`

Пример использования:
```http request
https://clientapi.adstat.pro/api/v1/advertising/telegram/upload_media?account_id=ACC0000222
```
:::
#### __Описание query-параметров__
:::info

| Параметр          | Тип     | Описание                  |
|-------------------|---------|---------------------------|
| `ad_id`           | Число   | Идентификатор объявления  |

:::

#### __Описание body__
:::info
В теле (body) HTTP-запроса нужно передать параметры, которые используются для создания или обновления информации о медиа-файле, связанном с объявлением.

Эти параметры должны быть отправлены в формате FormData. В данном случае, ожидается наличие следующих полей:

`media` — содержит сам медиа-файл, который требуется загрузить.
Другие параметры (например, идентификатор аккаунта) могут быть добавлены в запрос в зависимости от требований API.

Пример создания тела запроса на JavaScript: 

```js
const mediaFormData = new FormData();
mediaFormData.append("media", file);
```
:::

Вот как можно дополнить описание ответа в виде таблицы:

---

#### **Описание ответа**
:::info  
Пример ответа:
```json  
{  
    "media_token": "test_token",  
    "content_type": "image/jpeg",  
    "error": null  
}  
```
:::

#### Параметры успешного ответа:
:::info  

| Параметр       | Тип данных   | Описание                                                                 |
|----------------|--------------|---------------------------------------------------------------------------|
| `media_token`  | `string`     | Уникальный токен, идентифицирующий загруженный медиа-файл.               |
| `content_type` | `string`     | MIME-тип загруженного файла (например, `image/jpeg`, `video/mp4`).       |
| `error`        | `null` или `string` | Статус ошибки: `null`, если запрос успешен; строка с описанием ошибки в случае неудачи. |

Если запрос выполнен успешно, поле `error` всегда будет равно `null`.  
:::
